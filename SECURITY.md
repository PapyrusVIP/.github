# Security Policy

Thanks for helping make Papyrus more secure!

## Reporting a Vulnerability

**DO NOT** open a public issue, instead send your report privately to [security@papyrus.vip](mailto:security@papyrus.vip).

  * All Papyrus products and open source projects are in scope for reporting.

Reporter(s) can expect an inital response within 48 hours, acknowledging the issue was received.
